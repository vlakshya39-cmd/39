# 📄 PDF Splitter & Merger Studio

A modern, browser-based PDF utility that allows users to split and merge PDF files without uploading them to any server. Built as a single HTML application using HTML, CSS, JavaScript, PDF-Lib, and PDF.js, all processing happens locally for better privacy and performance.

---

## ✨ Features

### 📄 PDF Splitter
- Upload a PDF file
- Split by custom page range
- Extract individual pages
- Split every page into separate PDFs
- Download processed PDFs

### 📚 PDF Merger
- Upload multiple PDF files
- Merge PDFs into one document
- Maintain file order
- Download merged PDF

### 👀 PDF Preview
- Preview uploaded PDF pages
- Display page thumbnails
- Easy navigation

### 🎨 User Interface
- Modern Glassmorphism design
- Responsive layout
- Drag & Drop file upload
- Smooth animations
- Progress indicators
- Toast notifications
- Dark theme

## 🚀 How to Run

1. Download the repository.
2. Open `pdf-splitter-merger.html` in any modern web browser.
3. Upload your PDF file(s).
4. Choose either Split or Merge mode.
5. Process the PDF.
6. Preview the output.
7. Download the generated PDF.

No installation or backend server is required.
